# Reviews plugin for Monstra CMS

Links: [http://monstra.promo360.ru/plugin/reviews](http://monstra.promo360.ru/plugin/reviews)

## Usage
Open the link [http://example.org/reviews](http://example.org/reviews) after installing the plugin

## Installation
* Download the latest release 
* Extract the content of the zip file into the Monstra 'plugins' directory

## History
* v.1.0.0 - first public release

## Requirements
* Monstra CMS version 2.2.x or 2.3.x
